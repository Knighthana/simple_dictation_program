## 关于概率论与数理统计的听写

首先坚定信念，概率论与数理统计的几个公式没什么难背的

不要死记硬背，更不要光背字母不理解含义，要搞清楚一个公式有几个部分，分子分母有几项，都是什么含义，有无指数，指数是多少

做到这些，半个小时到一个小时的反复练习就足以记住了（说这个不是表示可以松懈状态拖到考前再背

如果你面临考试但还没有记住这些公式，看到这行之后就应该下载编译，找出你的笔记，开始背公式了

相信自己能通过考试的
